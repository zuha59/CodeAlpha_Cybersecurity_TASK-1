# Basic Network Sniffer

## Overview

This project is a Python-based network packet sniffer developed using the Scapy library.

It captures network packets and displays:

- Source IP Address
- Destination IP Address
- Protocol Type
- Port Numbers
- Packet Payload

## Technologies Used

- Python 3
- Scapy

## Installation

```bash
pip install scapy
```

## Run the Program

```bash
python network_sniffer.py
```

## Features

- Real-time packet capture
- TCP packet detection
- UDP packet detection
- ICMP packet detection
- Payload analysis
